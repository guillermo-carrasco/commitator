Welcome to Commitator!

With this web app, you can see some information about your GitHub organization in a very cool way! It will show you all kind of useful statistics from your repositories and members.

Try it out right now, select a time period and write the name of your organization on the box below, click on Fetch Data and enjoy!

Please note that due to how GitHub API works, and depending on your repositories activity, the longer the time period you select, the more API calls Commitator has to make to GitHub, and therefore the response time can be slower.

Pick up a time period (default is last 30 days)
